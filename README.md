📅 Ryan's Assignment Tracker
A custom Discord bot built to manage 8th-grade deadlines. It features a persistent Blue Dashboard that auto-cleans at 8:00 AM and a Red Reminder system that pings the class at 2:30 PM.

⚡ Quick Guide
Setup: Use /setup_tracker (Admin only) to launch the dashboard.

Add Tasks: Use the dropdown on the dashboard, then select Priority in the private follow-up message.

Logic: 🔴 High Priority tasks trigger an @everyone ping exactly 1 day before they are due.

Manual: Type /what for a quick in-app recap of all commands and rules.
